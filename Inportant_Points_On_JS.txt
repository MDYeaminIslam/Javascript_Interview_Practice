Question No. 1:
## What is Javascript ? What is the role of JS engines?

Ans: Javascript is a programming language that is used for converting static web pages to interective and dynamic web pages.

A Javascript engine is a program present in web browsers that executes JavaScript code.

Javascript engines:
V8 -> Chrome
Spider-Monkey -> FireFox
Javascript-Core -> Safari
Chakra -> Edge.
